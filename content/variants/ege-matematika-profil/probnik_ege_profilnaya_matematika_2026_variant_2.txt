1. Внутренний угол C равен 180° - 50° - 70° = 60°.
Внешний угол, смежный с ним, равен 180° - 60° = 120°.
Ответ: 120.

2. a - b = (1; 4) - (5; -2) = (-4; 6).
a + 2b = (1; 4) + (10; -4) = (11; 0).
Скалярное произведение: (-4) * 11 + 6 * 0 = -44.
Ответ: -44.

3. Объём прямоугольного параллелепипеда:
V = 5 * 6 * 7 = 210.
Ответ: 210.

4. Всего шаров 7 + 13 = 20.
Вероятность выбрать белый шар: 7 / 20 = 0,35.
Ответ: 0,35.

5. Вероятность того, что не произойдёт ни A, ни B:
(1 - 0,6)(1 - 0,5) = 0,4 * 0,5 = 0,2.
Значит вероятность хотя бы одного события:
1 - 0,2 = 0,8.
Ответ: 0,8.

Время на втором участке: 180 / 90 = 2 часа.

Средняя скорость: 360 / 5 = 72 км/ч.

sqrt(32) / sqrt(2) = sqrt(16) = 4.

12. y' = -3x^2 + 12x = -3x(x - 4).
Критические точки на [0; 5]: x = 0 и x = 4.
Проверим значения:
y(0) = 0,
y(4) = -64 + 96 = 32,
y(5) = -125 + 150 = 25.
Наибольшее значение равно 32.
Ответ: 32.

13. cos 2x = cos x.
Используем cos 2x = 2cos^2 x - 1:
2cos^2 x - 1 = cos x,
2cos^2 x - cos x - 1 = 0.
Пусть t = cos x:
2t^2 - t - 1 = 0,
(2t + 1)(t - 1) = 0.
t = 1 или t = -1/2.

На отрезке [0; 2π]:
cos x = 1 при x = 0 и x = 2π;
cos x = -1/2 при x = 2π/3 и x = 4π/3.

Ответ: 0; 2π/3; 4π/3; 2π.

Критерии: 2 балла — получены все корни на отрезке; 1 балл — верное преобразование, но ошибка в отборе; 0 баллов — решение неверно.

14. а) В равностороннем треугольнике ABC медиана AH является высотой, значит AH перпендикулярно BC.
Так как призма прямая, A1A перпендикулярна плоскости ABC. Проекция отрезка A1H на плоскость основания — AH. По теореме о трёх перпендикулярах A1H перпендикулярен BC. Следовательно, расстояние от точки A1 до прямой BC равно A1H.

б) В равностороннем треугольнике со стороной 6 высота AH равна 6sqrt(3)/2 = 3sqrt(3).
Так как A1A = 8 и A1A перпендикулярна плоскости основания:
A1H = sqrt(A1A^2 + AH^2) = sqrt(8^2 + (3sqrt(3))^2) = sqrt(64 + 27) = sqrt(91).

Ответ: sqrt(91).

Критерии: 3 балла — доказательство и вычисление выполнены полностью; 2 балла — верная идея, но доказательство неполное; 1 балл — верно найдена высота основания или составлена формула; 0 баллов — решение неверно.

15. Решим методом интервалов:
((x - 4)(x + 2)) / (x - 1) <= 0.

Критические точки: x = -2, x = 1, x = 4.
Точка x = 1 не входит в область определения.

Проверим интервалы:
x = -3: (-7)(-1)/(-4) < 0, подходит;
x = 0: (-4)(2)/(-1) > 0, не подходит;
x = 2: (-2)(4)/(1) < 0, подходит;
x = 5: (1)(7)/(4) > 0, не подходит.

Нули числителя включаем: x = -2 и x = 4.
Ответ: (-∞; -2] ∪ (1; 4].

Критерии: 2 балла — верный ответ с учётом области определения; 1 балл — верно найдены критические точки, но есть ошибка в знаках; 0 баллов — решение неверно.

16. Пусть ежегодный платёж равен x.

После первого января долг становится 455000 * 1,2 = 546000, после платежа: 546000 - x.
После второго года: (546000 - x) * 1,2 - x.
После третьего года долг должен стать 0.

Используем формулу:
x(1 + 1,2 + 1,2^2) = 455000 * 1,2^3.
1 + 1,2 + 1,44 = 3,64.
455000 * 1,728 = 786240.
x = 786240 / 3,64 = 216000.

Проверка:
455000 * 1,2 - 216000 = 330000;
330000 * 1,2 - 216000 = 180000;
180000 * 1,2 - 216000 = 0.

Ответ: 216000.

Критерии: 2 балла — составлена верная финансовая модель и найден ответ; 1 балл — модель верная, но вычисления с ошибкой; 0 баллов — решение неверно.

17. а) Пусть M — середина гипотенузы AB. В прямоугольном треугольнике середина гипотенузы равноудалена от всех трёх вершин, поэтому CM = AM = BM = AB/2. Значит медиана CM равна половине гипотенузы.

б) Гипотенуза:
AB = sqrt(9^2 + 12^2) = sqrt(225) = 15.
Радиус вписанной окружности прямоугольного треугольника:
r = (a + b - c) / 2 = (9 + 12 - 15) / 2 = 3.

Ответ: 3.

Критерии: 3 балла — доказательство и вычисление выполнены полностью; 2 балла — верное вычисление при неполном доказательстве; 1 балл — найдена гипотенуза или записана формула радиуса; 0 баллов — решение неверно.

18. Рассмотрим график функции y = |x^2 - 4|.
При a < 0 решений нет.
При a = 0 уравнение |x^2 - 4| = 0 имеет два корня: x = -2 и x = 2.
При 0 < a < 4 горизонтальная прямая y = a пересекает график в четырёх точках.
При a = 4:
|x^2 - 4| = 4.
x^2 - 4 = 4 -> x = ±2sqrt(2);
x^2 - 4 = -4 -> x = 0.
Итого три корня.
При a > 4 решений два.

Ответ: a = 4.

Критерии: 4 балла — рассмотрены все случаи и получено a = 4; 3 балла — верная идея, но неполно описаны крайние случаи; 2 балла — найдено a = 4 без полного обоснования; 1 балл — построена верная логика по графику; 0 баллов — решение неверно.

19. Дано:
1/x + 1/y = 1/6.
Получаем:
6x + 6y = xy,
xy - 6x - 6y = 0,
xy - 6x - 6y + 36 = 36,
(x - 6)(y - 6) = 36.

При x <= y перебираем пары делителей 36:
1 и 36, 2 и 18, 3 и 12, 4 и 9, 6 и 6.

Получаем:
(7; 42), (8; 24), (9; 18), (10; 15), (12; 12).

Ответ: (7; 42), (8; 24), (9; 18), (10; 15), (12; 12).

Критерии: 4 балла — полный перебор и все пары; 3 балла — пропущена одна пара; 2 балла — верное преобразование, но перебор неполный; 1 балл — выполнено начальное преобразование; 0 баллов — решение неверно.
