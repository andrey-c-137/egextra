1. Так как AB = AC, углы при основании равны: угол B = угол C.
Внешний угол при C равен 112°, значит внутренний угол C равен 180° - 112° = 68°.
Тогда угол B = 68°, угол A = 180° - 68° - 68° = 44°.
Ответ: 44.

2. 2a - b = (4; -2) - (-3; 4) = (7; -6).
a + b = (2; -1) + (-3; 4) = (-1; 3).
Скалярное произведение: 7 * (-1) + (-6) * 3 = -7 - 18 = -25.
Ответ: -25.

3. Диагональ прямоугольного параллелепипеда:
d = sqrt(3^2 + 4^2 + 12^2) = sqrt(9 + 16 + 144) = sqrt(169) = 13.
Ответ: 13.

4. Всего карандашей 12 + 18 = 30. Вероятность выбрать красный:

Ответ: 0,4.

5. События независимы, поэтому вероятность совместной работы:

12. y' = 12 - 3x^2.
y' = 0 при x = 2 и x = -2. На отрезке [0; 3] подходит x = 2.
Проверим значения:

y(2) = 24 - 8 = 16,

Ответ: 16.

13. 2sin^2 x - sin x - 1 = 0.
Пусть t = sin x. Тогда 2t^2 - t - 1 = 0.
(2t + 1)(t - 1) = 0, значит t = 1 или t = -1/2.

На отрезке [0; 2π]:
sin x = 1 при x = π/2.
sin x = -1/2 при x = 7π/6 или x = 11π/6.

Ответ: π/2; 7π/6; 11π/6.

Критерии: 2 балла — получены все корни на указанном отрезке; 1 балл — верно решено квадратное уравнение относительно sin x, но есть ошибка в отборе корней; 0 баллов — решение неверно или отсутствует.

14. а) В правильной четырёхугольной пирамиде вершина S проектируется в центр основания O. В квадрате ABCD центр O лежит на диагонали AC и равноудалён от A и C. Так как SO перпендикулярна плоскости основания, получаем:
SA^2 = SO^2 + OA^2,
SC^2 = SO^2 + OC^2.
OA = OC, значит SA = SC. Следовательно, треугольник SAC равнобедренный.

б) Расстояние от точки O до стороны AB квадрата равно половине стороны: 3.
Пусть H — середина AB. Тогда OH перпендикулярно AB, SO перпендикулярно плоскости ABCD. По теореме о трёх перпендикулярах SH перпендикулярно AB, значит SH — искомое расстояние.
SH = sqrt(SO^2 + OH^2) = sqrt(4^2 + 3^2) = 5.

Ответ: 5.

Критерии: 3 балла — доказательство и вычисление выполнены полностью; 2 балла — верная идея и вычисление, но доказательство неполное; 1 балл — найден один ключевой элемент, например OH = 3 или применена верная формула расстояния; 0 баллов — решение неверно.

15. Разложим числитель:
x^2 - 5x + 6 = (x - 2)(x - 3).
Решаем неравенство
((x - 2)(x - 3)) / (x - 1) >= 0.

Критические точки: x = 1, x = 2, x = 3.
Точка x = 1 не входит в область определения, точки x = 2 и x = 3 являются нулями числителя.

Проверим знаки на интервалах:
x = 0: (0 - 2)(0 - 3)/(0 - 1) = 6 / (-1) < 0, не подходит.
x = 1,5: (1,5 - 2)(1,5 - 3)/(1,5 - 1) > 0, подходит.
x = 2,5: (2,5 - 2)(2,5 - 3)/(2,5 - 1) < 0, не подходит.
x = 4: (4 - 2)(4 - 3)/(4 - 1) > 0, подходит.

С учётом знака >= 0 нули числителя включаются.
Ответ: (1; 2] ∪ [3; +∞).

Критерии: 2 балла — верный ответ с учётом области определения; 1 балл — верно найдены критические точки, но допущена ошибка в одном интервале; 0 баллов — решение неверно.

16. Пусть ежегодно вкладчик снимает x рублей.

После первого года:
331000 * 1,1 - x = 364100 - x.

После второго года:
(364100 - x) * 1,1 - x.

После третьего года:
((364100 - x) * 1,1 - x) * 1,1 - x = 0.

Удобнее раскрыть через прогрессию:
x(1 + 1,1 + 1,1^2) = 331000 * 1,1^3.
1 + 1,1 + 1,21 = 3,31.
331000 * 1,331 = 440561.
x = 440561 / 3,31 = 133100.

Проверка:
331000 * 1,1 - 133100 = 231000;
231000 * 1,1 - 133100 = 121000;
121000 * 1,1 - 133100 = 0.

Ответ: 133100.

Критерии: 2 балла — составлена верная модель и получен ответ; 1 балл — модель составлена верно, но есть вычислительная ошибка; 0 баллов — модель неверна.

17. а) Пусть M — середина BC. Так как AB = AC, треугольник ABC равнобедренный. В равнобедренном треугольнике медиана, проведённая к основанию, является высотой. Значит AM перпендикулярна BC.

б) BM = MC = 5. В прямоугольном треугольнике ABM:
AM = sqrt(AB^2 - BM^2) = sqrt(13^2 - 5^2) = sqrt(169 - 25) = 12.
Площадь треугольника ABC:
S = 1/2 * BC * AM = 1/2 * 10 * 12 = 60.
Полупериметр:
p = (13 + 13 + 10) / 2 = 18.
Радиус вписанной окружности:
r = S / p = 60 / 18 = 10/3.

Ответ: 10/3.

Критерии: 3 балла — доказательство и вычисление радиуса выполнены полностью; 2 балла — решение в целом верное, но есть неточность в доказательстве или вычислениях; 1 балл — найдена высота или площадь; 0 баллов — решение неверно.

18. Рассмотрим квадратное уравнение:
x^2 - 2(a + 1)x + a^2 - 1 = 0.

Дискриминант:
D = [-2(a + 1)]^2 - 4(a^2 - 1)
= 4(a + 1)^2 - 4a^2 + 4
= 4(a^2 + 2a + 1 - a^2 + 1)
= 8(a + 1).

Корни существуют при a >= -1.
Произведение корней равно a^2 - 1.
Чтобы был ровно один положительный корень, возможны случаи:
1) корни разных знаков: a^2 - 1 < 0, то есть -1 < a < 1;
2) один корень равен 0, другой положительный: a^2 - 1 = 0.

При a = 1 уравнение x^2 - 4x = 0, корни 0 и 4 — ровно один положительный.
При a = -1 уравнение x^2 = 0, положительных корней нет.

Итого a ∈ (-1; 1].

Ответ: (-1; 1].

Критерии: 4 балла — полное рассмотрение дискриминанта, произведения корней и граничных случаев; 3 балла — верная идея, но один граничный случай описан неполно; 2 балла — найдена основная область -1 < a < 1 без корректной проверки границ; 1 балл — записаны формулы Виета или дискриминант; 0 баллов — решение неверно.

19. Дано:
1/x + 1/y = 1/6.
Приведём к общему знаменателю:
(x + y) / xy = 1/6,
6x + 6y = xy,
xy - 6x - 6y = 0.

Добавим 36:
xy - 6x - 6y + 36 = 36,
(x - 6)(y - 6) = 36.

Так как x и y натуральные и x <= y, получаем x - 6 <= y - 6.
Нужно перебрать пары положительных делителей числа 36:
1 * 36, 2 * 18, 3 * 12, 4 * 9, 6 * 6.

Отсюда:
x - 6 = 1, y - 6 = 36 -> (7; 42);
x - 6 = 2, y - 6 = 18 -> (8; 24);
x - 6 = 3, y - 6 = 12 -> (9; 18);
x - 6 = 4, y - 6 = 9 -> (10; 15);
x - 6 = 6, y - 6 = 6 -> (12; 12).

Ответ: (7; 42), (8; 24), (9; 18), (10; 15), (12; 12).

Критерии: 4 балла — верное преобразование и полный перебор пар; 3 балла — верное преобразование, но пропущена одна пара; 2 балла — получено уравнение с делителями, но перебор неполный; 1 балл — выполнено начальное преобразование; 0 баллов — решение неверно.
