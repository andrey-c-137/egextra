14. а) В прямоугольном параллелепипеде боковые рёбра AA1 и CC1 перпендикулярны плоскости основания ABCD, следовательно, AA1 параллельно CC1.
Отрезки AC и A1C1 — диагонали равных параллельных оснований, поэтому AC параллельно A1C1.
Четырёхугольник ACC1A1 имеет две пары параллельных сторон, значит он параллелограмм.
Так как AA1 перпендикулярна плоскости основания, AA1 перпендикулярна AC. Следовательно, параллелограмм ACC1A1 имеет прямой угол, то есть является прямоугольником.

б) Диагональ основания:
AC = 6sqrt(2).
Высота прямоугольника:
AA1 = 8.
Площадь сечения:
S = AC * AA1 = 6sqrt(2) * 8 = 48sqrt(2).

Ответ: 48sqrt(2).

Критерии: 3 балла — доказательство и вычисление выполнены полностью; 2 балла — верно найдено сечение и площадь, но доказательство неполное; 1 балл — найдена диагональ основания; 0 баллов — решение неверно.

15. Решаем:
((x - 1)(x + 4)) / (x - 3)^2 > 0.

Область определения: x != 3.
Так как (x - 3)^2 > 0 при x != 3, знак дроби определяется числителем:
(x - 1)(x + 4) > 0.

Произведение положительно при x < -4 или x > 1.
При этом x = 3 нужно исключить.

Ответ: (-∞; -4) ∪ (1; 3) ∪ (3; +∞).

Критерии: 2 балла — верный ответ с исключением x = 3; 1 балл — верно решено неравенство числителя, но забыто исключить x = 3; 0 баллов — решение неверно.

16. Пусть ежегодно снимают x рублей.

17. а) В ромбе все стороны равны. Диагонали ромба являются биссектрисами углов и делят друг друга пополам. Также треугольники, на которые диагонали делят ромб, имеют равные стороны. Из равенства соответствующих треугольников следует, что углы при точке пересечения диагоналей равны. Сумма смежных углов равна 180°, значит каждый из них равен 90°. Следовательно, диагонали ромба взаимно перпендикулярны.

б) Пусть O — точка пересечения диагоналей. Тогда AO = AC/2 = 8.
В прямоугольном треугольнике AOB гипотенуза AB = 10.
BO = sqrt(AB^2 - AO^2) = sqrt(10^2 - 8^2) = sqrt(36) = 6.
Значит BD = 2BO = 12.
Площадь ромба:
S = (AC * BD) / 2 = (16 * 12) / 2 = 96.

Ответ: 96.

Критерии: 3 балла — доказательство и вычисление выполнены полностью; 2 балла — площадь найдена верно, доказательство неполное; 1 балл — найдена вторая диагональ; 0 баллов — решение неверно.

18. Уравнение:
x^2 - 2ax + a + 2 = 0.

Чтобы было два различных положительных корня, нужны условия:
1) D > 0;
2) сумма корней положительна;
3) произведение корней положительно.

По теореме Виета:
x1 + x2 = 2a,
x1x2 = a + 2.

Условия:
2a > 0 -> a > 0;
a + 2 > 0 -> a > -2.

Дискриминант:
D = (-2a)^2 - 4(a + 2) = 4a^2 - 4a - 8 = 4(a^2 - a - 2) = 4(a - 2)(a + 1).
D > 0 при a < -1 или a > 2.

Совмещаем с a > 0:
a > 2.

Ответ: (2; +∞).

Критерии: 4 балла — все условия для двух положительных корней рассмотрены; 3 балла — верный ответ при неполном обосновании одного условия; 2 балла — найдены условия Виета, но дискриминант рассмотрен неполно; 1 балл — записаны формулы Виета или дискриминант; 0 баллов — решение неверно.

19. Дано:
xy - 3x - 2y = 6.
Добавим 6 к обеим частям:
xy - 3x - 2y + 6 = 12.
Левая часть раскладывается:
(x - 2)(y - 3) = 12.

Так как x и y натуральные, x - 2 и y - 3 — положительные делители 12. Переберём:
1 * 12 -> x = 3, y = 15;
2 * 6 -> x = 4, y = 9;
3 * 4 -> x = 5, y = 7;
4 * 3 -> x = 6, y = 6;
6 * 2 -> x = 8, y = 5;
12 * 1 -> x = 14, y = 4.

Ответ: (3; 15), (4; 9), (5; 7), (6; 6), (8; 5), (14; 4).

Критерии: 4 балла — верное преобразование и полный перебор; 3 балла — пропущена одна пара; 2 балла — верное преобразование, но перебор неполный; 1 балл — начато преобразование уравнения; 0 баллов — решение неверно.
